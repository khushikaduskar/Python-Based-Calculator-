# Calculator Application

## Project Description
This project is a simple Calculator Application developed using Python.  
It performs basic arithmetic operations such as addition, subtraction, multiplication, and division through a user-friendly interface.

The project is created as part of academic coursework to understand Python programming, logic building, and GUI development.

---

## Objectives
- To develop a basic calculator using Python
- To understand event handling and functions
- To implement arithmetic operations programmatically
- To gain hands-on experience with Python GUI / Jupyter Notebook

---

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear / Reset option
- Simple and easy-to-use interface

---

## Technologies Used
- Python
- Tkinter (for GUI) / Jupyter Notebook
- Basic Python Libraries

---

## How to Run the Project
1. Install Python on your system
2. Download or clone this repository
3. Open the project folder
4. Run the file:
   - `calculator.ipynb` using Jupyter Notebook  
   **OR**
   - `calculator.py` using Python

---

## Output
The calculator accepts user input and displays the calculated result instantly.
